Geospatial Data Collection Project

The Geospatial Data Collection project includes extensive datasets critical for environmental and agricultural research in South Africa. This collection contains:

  • CHIRPS rainfall data collection at ward level.
  • High Resolution Settlement Layer data collection.
  • South African National land cover data collection.
  • Moderate-resolution Imaging Spectroradiometer (MODIS) data collection.
  • Crop classification using Landsat8 and Sentinel2 satellite images for machine learning applications.
