End to End Trajectories Modelling for Type 2 Diabetes Complications

The End to End Trajectories Modelling project focuses on type 2 diabetes complications. Key steps involved:

  • Extracting relevant information from EHR data.
  • Extracting trajectories and their likelihoods using methods like Markov chain and PrefixSpan.
  • Writing a report on the results to facilitate better healthcare decisions.

Geospatial Data Collection Project

The Geospatial Data Collection project includes datasets for environmental and agricultural research in South Africa. This collection contains:

  • CHIRPS rainfall data collection at ward level.
  • High Resolution Settlement Layer data collection.
  • South African National land cover data collection.
  • Moderate-resolution Imaging Spectroradiometer (MODIS) data collection.
  • Crop classification using Landsat8 and Sentinel2 satellite images for machine learning applications.

Lake Chad Storage Forecasting

The Lake Chad Storage Forecasting project is aimed at forecasting water storage in the Lake Chad Basin. It includes:

  • An exploratory data analysis (EDA) folder for initial data insights.
  • LSTM models that were used to predict the storage outputs.
