Welcome to my Portfolio

Here is a selection of my most notable projects in Data Science:

Geospatial Data Collection Project

This project explains the data collection process through the python api of google earth engine for agricultural economics studies. The data sources include:

  • CHIRPS rainfall data collection at ward level.
  • High Resolution Settlement Layer data collection.
  • South African National land cover data collection.
  • Moderate-resolution Imaging Spectroradiometer (MODIS) data collection.
  • Crop classification using Landsat8 and Sentinel2 satellite images.

Absenteeism Prediction

This project aims to identify the keys determining factors of absenteism in a company. In this project we develop and deploy a machine learning model that predicts absenteeism. The model helps in identifying the key factors contributing to employee absenteeism and assists HR in making informed decisions.

Lake Chade storage forecasting

In this project we develops and deploy a LSTM for storage forecasting in the Lake Chad Basin. This project is part of a bigger project on optimal water resources management in the Lake Chad Basin.
